A comprehensive digital notice board system for Marathwada Mitra Mandal's College of Engineering (MMCOE) to manage and display college notices, announcements, and events digitally.

## ✨ Features

### 👥 Multi-Role System
- **Admin Dashboard**: Full control over users, notices, and categories
- **Faculty Portal**: Post notices with scheduling and attachment support
- **Student View**: Browse and search notices with category filtering

### 📋 Notice Management
- ✅ Create, read, update, and delete notices
- 📅 Schedule notices for future posting
- ⏰ Auto-expiry of notices with history archive
- 📎 File attachments (PDF, DOC, JPG, PNG, TXT)
- 🏷️ Category-based organization
- ⚡ Priority tagging (Normal/Urgent)

### 🔐 Security Features
- Password hashing with bcrypt
- Security questions for password recovery
- Session management
- Faculty approval workflow
- XSS protection with input sanitization

### 🎨 User Interface
- Responsive design (mobile, tablet, desktop)
- MMCOE branding (Maroon & Gold theme)
- Smooth animations and hover effects
- Modal views for notice details
- Year-wise notice history filtering
